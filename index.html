Biography

I received my M.Eng degree from East China Normal University in Dec, 2019, supervised by prof. Faming Fang. I have done some work on computation photography tasks using traditional optimization based methods. Now I’m looking forward to use more efficient approaches (e.g. learning-based) to solve more realistic problems.
